About

The Secure Stash App is a secure and easily accessible repository for sensitive user files and documents.

Encrypted

All files are secured with military grade AES encryption and have unique keys generated on a per file basis.

The Only Downside is Complete Ownership

The user has the ultimate control and ownership over all files. The app simply serves as an encryption medium for sensitive user data, This means:

  • All information is stored and encrypted locally on YOUR device.
  • Files/encryption keys are NOT stored in remote locations, such as the cloud.
  • File access passwords are hashed and stored locally in an encrypted Database
